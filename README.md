## 快速入门

1. **克隆或下载** 本仓库到本地。
2. 将 `components` 目录中的 `table` 组件复制到你的小程序项目的 `components` 文件夹内。
3. 在需要使用此组件的页面JSON文件中引入组件：
   ```json
   {
     "usingComponents": {
       "custom-table": "./components/table/table"
     }
   }
   ```
4. 在WXML文件中使用组件，并传入相应的数据：
   ```xml
   <custom-table binddata="yourData" />
   ```
5. 在对应的JS文件中处理数据与事件。
6. 参考提供的教程完成配置与定制。
